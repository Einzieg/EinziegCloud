# EinziegCloud

## 多人共享云服务

## EinziegCloud is a cloud storage service that allows you to store your files in the cloud and access them from anywhere.

## 技术栈

### 后端
#### Java 21

#### SpringBoot 3.2
#### SpringSecurity 6

#### Jjwt 0.11

#### Redis 3.2

#### MySQL 8.0.32

### 前端

#### Vue 3
#### VITE
#### TypeScript

### 监控：Spring Boot Actuator + Prometheus + Grafana
